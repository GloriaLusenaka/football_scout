# EA Eye Scout Predictor

A data-driven football scouting tool that predicts player potential and market value to uncover undervalued talent in East Africa.

## What It Does

Enter basic player performance metrics and get instant scout insights:

- Potential Score (0-100)
- Estimated Market Value (€)
- Scout Recommendation (Trial / Watchlist / Monitor)
- Undervalued Gem Alert

## How It Works

The tool uses a Random Forest machine learning model trained on 500+ East African player profiles. It compares a player's metrics against regional averages and predicts their future value.

## Features

- Player potential prediction
- Market value estimation
- Radar chart comparing player vs regional benchmarks
- Hidden gems leaderboard showing most undervalued players
- Clear scout action items

## Tech Stack

- Python
- Streamlit
- Scikit-learn (Random Forest)
- Pandas & NumPy
- Plotly
